update.01 - make more modern, bring up to date

ARGs 2 and 3 added based on evidence in THYME-AMR. Compare/contrast CT-scan example with: the website is updating, where website is ARG1; the website is becoming modernized/updated. In updated CT-scan, the scan itself is the thing constituting the update; it's the newest version. For these cases, ARG1 is unlikely to be explicit; it seems it would only be filled by a generic reference to the type of scan (something like, 'In terms of CT scans, we'll get an updated one'?? Again, unlikely to occur).
